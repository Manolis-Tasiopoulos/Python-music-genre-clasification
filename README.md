## Εγκατάσταση
  * Clone the repository and then:
    * User interface instalation
	```sh
	$ cd Python-music-genre-clasification	
	$ npm install
	```
    * Python libraries installation
	```sh
	$ $ pip install -r requirements.txt	
	```
  * Run
	```sh
	$ $ npm start	
	```
